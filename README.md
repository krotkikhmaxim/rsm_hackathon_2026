# Система прогнозирования киберугроз

**MVP-прототип для хакатона**

## О проекте

Интеллектуальная система для прогнозирования кибератак на предприятия. Анализирует данные о прошлых инцидентах, учитывает отрасль, регион, размер инфраструктуры и другие параметры, чтобы предсказать вероятность атаки, её временные характеристики, тип угрозы, объект воздействия и уровень уязвимости. На основе прогноза система формирует приоритетные рекомендации по защите.

## Ключевые возможности

- **6 прогнозных моделей**: вероятность атаки, время атаки, тип угрозы, объект воздействия, уровень уязвимости, рекомендации.
- **SHAP-объяснения**: визуальное обоснование каждого прогноза (почему модель приняла такое решение).
- **Демо-сценарии**: предустановленные примеры для быстрой демонстрации (критический / высокий / низкий риск).
- **Каталог угроз ФСТЭК**: 227 угроз с фильтрацией и поиском.
- **Аналитический дашборд**: интерактивные графики инцидентов по регионам, отраслям и времени.
